1. Pengaturan Ekstensi Visual Studio Code (VS Code)

Untuk memulai, pastikan Anda sudah menginstal Visual Studio Code (VS Code). VS Code adalah code editor populer yang akan sangat membantu dalam menulis dan mengelola kode JavaScript Anda.

Berikut adalah ekstensi VS Code yang sangat direkomendasikan untuk pengembangan JavaScript:

  • Prettier - Code formatter: Ini akan merapikan kode Anda secara otomatis, memastikan kode Anda selalu terlihat bersih dan profesional.
  • ESLint: Ini seperti asisten pribadi yang memeriksa kode JavaScript Anda untuk mencari potensi masalah, kesalahan, atau pelanggaran gaya penulisan kode.
  • Live Server: Berguna untuk melihat hasil file HTML/CSS/JavaScript Anda langsung di browser. Anda tidak perlu me-refresh halaman secara manual.
  • JavaScript (ES6) code snippets: Ekstensi ini menyediakan potongan-potongan kode JavaScript yang sering digunakan. Ini akan mempercepat penulisan kode Anda.

Cara menginstal ekstensi:

  1. Buka VS Code.
  2. Klik ikon Extensions di sidebar kiri (terlihat seperti empat kotak kecil).
  3. Di kolom pencarian, ketik nama ekstensi (misalnya: "Prettier").
  4. Klik tombol Install di samping nama ekstensi yang muncul.

3. Praktik Pembuatan Variabel

Apa persamaan dan perbedaan let, const, dan var?

Di JavaScript, var, let, dan const adalah kata kunci yang kita gunakan untuk mendeklarasikan variabel. Ketiganya punya aturan main yang berbeda dalam hal cakupan (scope) dan kemampuan untuk diubah (mutability).

Fitur / Kata Kunci var let const
Cakupan (Scope) Function-scoped Block-scoped Block-scoped
Redeklarasi Bisa (dideklarasikan ulang) Tidak bisa (akan error) Tidak bisa (akan error)
Reassign (Ubah Nilai) Bisa (nilai bisa diubah) Bisa (nilai bisa diubah) Tidak bisa (nilai harus tetap)
Hoisting Di-hoist & inisialisasi undefined Di-hoist tapi tidak diinisialisasi Di-hoist tapi tidak diinisialisasi

Praktikkan Contoh Penggunaan Variabel Tersebut

Contoh Penggunaan var (Sebaiknya Dihindari)


// Deklarasi dan inisialisasi
var namaDepan = "Amay";
console.log(`var namaDepan (awal): ${namaDepan}`); // Output: var namaDepan (awal): Amay

// Reassign (ubah nilai) diizinkan
namaDepan = "Marvel";
console.log(`var namaDepan (reassign): ${namaDepan}`); // Output: var namaDepan (reassign): Marvel

// Redeklarasi diizinkan (ini bisa jadi sumber bug!)
var namaDepan = "Akbar";
console.log(`var namaDepan (redeklarasi): ${namaDepan}`); // Output: var namaDepan (redeklarasi): Akbar

// Contoh function-scoped:
if (true) {
  var jumlahProduk = 50;
}
console.log(`var jumlahProduk di luar blok: ${jumlahProduk}`); // Output: var jumlahProduk di luar blok: 50
// Variabel 'jumlahProduk' masih bisa diakses di luar blok 'if' karena 'var' bersifat function-scoped.

Contoh Penggunaan let


// Deklarasi dan inisialisasi
let suhu = 28;
console.log(`let suhu (awal): ${suhu}`); // Output: let suhu (awal): 28

// Reassign (ubah nilai) diizinkan
suhu = 30;
console.log(`let suhu (reassign): ${suhu}`); // Output: let suhu (reassign): 30

// Redeklarasi TIDAK diizinkan (akan menimbulkan error)
// let suhu = 25; // ERROR: Cannot redeclare block-scoped variable 'suhu'.
// console.log(suhu);

// Contoh block-scoped:
if (true) {
  let nilaiLetBlok = 123;
  console.log(`let nilaiLetBlok di dalam blok: ${nilaiLetBlok}`); // Output: let nilaiLetBlok di dalam blok: 123
}
// console.log(`let nilaiLetBlok di luar blok: ${nilaiLetBlok}`); // ERROR: nilaiLetBlok is not defined
// Variabel 'nilaiLetBlok' hanya bisa diakses di dalam blok 'if' karena 'let' bersifat block-scoped.

Contoh Penggunaan const


// Deklarasi dan inisialisasi (wajib langsung diberi nilai)
const PI_VALUE = 3.14159;
console.log(`const PI_VALUE (awal): ${PI_VALUE}`); // Output: const PI_VALUE (awal): 3.14159

// Reassign TIDAK diizinkan (akan menimbulkan error)
// PI_VALUE = 3.14; // ERROR: Assignment to constant variable.
// console.log(PI_VALUE);

// Redeklarasi TIDAK diizinkan (akan menimbulkan error)
// const PI_VALUE = 22/7; // ERROR: Cannot redeclare block-scoped variable 'PI_VALUE'.
// console.log(PI_VALUE);

// Contoh dengan objek (properti objek bisa diubah)
const itemKeranjang = {
  nama: "Baju Kaos",
  jumlah: 2,
  harga: 75000
};
console.log(`const itemKeranjang (awal):`, itemKeranjang); // Output: { nama: 'Baju Kaos', jumlah: 2, harga: 75000 }

itemKeranjang.jumlah = 3; // Properti objek 'jumlah' bisa diubah
console.log(`const itemKeranjang (jumlah diubah):`, itemKeranjang); // Output: { nama: 'Baju Kaos', jumlah: 3, harga: 75000 }

// Mengganti seluruh objek TIDAK diizinkan
// itemKeranjang = { nama: "Celana", jumlah: 1, harga: 150000 }; // ERROR: Assignment to constant variable.
// console.log(itemKeranjang);

4. Praktik Pembuatan Tipe Data

Tipe Data Primitif (Nilai Dasar)

Tipe data primitif adalah bentuk data paling dasar dalam JavaScript yang merepresentasikan satu nilai tunggal dan tidak memiliki properti atau metode.

Tipe Data: String (Teks)

Digunakan untuk menyimpan teks atau rangkaian karakter.


// Menggunakan double quotes
let namaProduk = "Laptop Gaming";
console.log(`Nama Produk: ${namaProduk}`); // Output: Nama Produk: Laptop Gaming

// Menggunakan single quotes
let deskripsiSingkat = 'Kuat dan Cepat!';
console.log(`Deskripsi: ${deskripsiSingkat}`); // Output: Deskripsi: Kuat dan Cepat!

// Menggunakan backticks (Template Literals) untuk embedding variabel
const merek = "ASUS";
let pesanLengkap = `Produk ${namaProduk} dari ${merek} sangat ${deskripsiSingkat.toLowerCase()}.`;
console.log(`Pesan Lengkap: ${pesanLengkap}`);
// Output: Produk Laptop Gaming dari ASUS sangat kuat dan cepat!.

console.log(`Tipe data namaProduk: ${typeof namaProduk}`); // Output: string

Tipe Data: Number (Angka)

Digunakan untuk menyimpan nilai numerik, baik bilangan bulat maupun desimal.


// Bilangan bulat (Integer)
let jumlahPengguna = 12345;
console.log(`Jumlah Pengguna: ${jumlahPengguna}`); // Output: Jumlah Pengguna: 12345

// Bilangan desimal (Float)
let ratingBintang = 4.7;
console.log(`Rating Bintang: ${ratingBintang}`); // Output: Rating Bintang: 4.7

// Operasi matematika
let hargaSatuan = 150000;
let kuantitas = 3;
let totalPembayaran = hargaSatuan * kuantitas;
console.log(`Total Pembayaran: Rp${totalPembayaran}`); // Output: Total Pembayaran: Rp450000

console.log(`Tipe data totalPembayaran: ${typeof totalPembayaran}`); // Output: number

Tipe Data: Boolean (Benar atau Salah)

Digunakan untuk menyimpan nilai logika, yaitu true atau false.


// Nilai true
let isLoggedIn = true;
console.log(`Pengguna sudah login? ${isLoggedIn}`); // Output: Pengguna sudah login? true

// Nilai false
let isAdmin = false;
console.log(`Pengguna adalah admin? ${isAdmin}`); // Output: Pengguna adalah admin? false

console.log(`Tipe data isLoggedIn: ${typeof isLoggedIn}`); // Output: boolean

Tipe Data Non-Primitif / Objek (Nilai Kompleks)

Tipe data non-primitif (atau objek) digunakan untuk menyimpan koleksi data yang lebih kompleks, seringkali berupa pasangan key-value atau daftar item.

Tipe Data: Array (Daftar)

Digunakan untuk menyimpan koleksi item yang berurutan, bisa berisi berbagai tipe data.


// Array daftar buah
let daftarBuah = ["Apel", "Pisang", "Ceri"];
console.log(`Daftar Buah: ${daftarBuah.join(', ')}`); // Output: Daftar Buah: Apel, Pisang, Ceri

// Mengakses elemen array berdasarkan indeks (mulai dari 0)
console.log(`Buah pertama: ${daftarBuah[0]}`); // Output: Buah pertama: Apel

// Array dengan tipe data campuran
let campuranData = ["Text", 123, true];
console.log(`Data Campuran: ${campuranData}`); // Output: Data Campuran: Text,123,true

console.log(`Tipe data daftarBuah: ${typeof daftarBuah}`); // Output: object (array secara teknis adalah object)

Tipe Data: Object (Objek)

Digunakan untuk menyimpan koleksi pasangan key-value (properti dan nilainya).


// Objek data pengguna
let dataPengguna = {
    nama: "Fitriani",
    umur: 24,
    isAktif: true
};
console.log(`Data Pengguna: ${JSON.stringify(dataPengguna)}`);
// Output: Data Pengguna: {"nama":"Fitriani","umur":24,"isAktif":true}

// Mengakses properti objek
console.log(`Nama Pengguna: ${dataPengguna.nama}`); // Output: Nama Pengguna: Fitriani

// Objek berlapis (Nested Object)
let infoKontak = {
    email: "fitriani@example.com",
    telepon: "08123456789",
    alamat: {
        jalan: "Jl. Anggrek No. 5",
        kota: "Bandung"
    }
};
console.log(`Kota Kontak: ${infoKontak.alamat.kota}`); // Output: Kota Kontak: Bandung

console.log(`Tipe data dataPengguna: ${typeof dataPengguna}`); // Output: object

Tipe Data: Null (Kosong yang Disengaja)

Menunjukkan ketiadaan nilai secara sengaja.


let selectedItem = null; // Variabel sengaja tidak memiliki nilai
console.log(`Nilai selectedItem: ${selectedItem}`); // Output: Nilai selectedItem: null

console.log(`Tipe data selectedItem: ${typeof selectedItem}`); // Output: object (ini adalah bug historis di JavaScript)

Tipe Data: Undefined (Belum Didefinisikan)

Menunjukkan bahwa sebuah variabel telah dideklarasikan tetapi belum diberi nilai, atau properti objek yang tidak ada.


let variabelTanpaNilai; // Variabel dideklarasikan tanpa inisialisasi
console.log(`Nilai variabelTanpaNilai: ${variabelTanpaNilai}`); // Output: Nilai variabelTanpaNilai: undefined

// Properti objek yang tidak ada
let objekKosong = {};
console.log(`Properti 'nonExistent' di objekKosong: ${objekKosong.nonExistent}`); // Output: Properti 'nonExistent' di objekKosong: undefined

console.log(`Tipe data variabelTanpaNilai: ${typeof variabelTanpaNilai}`); // Output: undefined
